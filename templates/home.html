🚀 GCloud Manager API

API para gerenciar buckets e arquivos no Google Cloud Storage.

Facilite o gerenciamento de buckets e arquivos com endpoints poderosos.

📄 Documentação Swagger

📋 Gerencie seus buckets e arquivos.

🔐 Autenticar 📂 Gerenciar Buckets

📋 Principais rotas

Endpoints Disponíveis
  • GET / - Página inicial da API
  • POST /auth - Autenticar credenciais
  • GET /buckets - Listar todos os buckets
  • POST /buckets - Criar um novo bucket
  • DELETE /buckets/<bucket_name> - Deletar um bucket
  • GET /buckets/<bucket_name>/files - Listar arquivos de um bucket
  • GET /<bucket_name>/files/<file_name> - Gerar link temporário
  • DELETE /<bucket_name>/files/<file_name> - Deletar um arquivo
  • GET /<bucket_name>/files/<file_name>/download - Fazer download de um arquivo
